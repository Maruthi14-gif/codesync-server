# codesync-server

This is the backend collaborative real-time code editor server.

## Stack
- Node.js + TypeScript
- WebSockets (`ws`)
- Yjs + `y-websocket` (custom integration)
- persistence (`y-leveldb`)
- AI complete (`@anthropic-ai/sdk`)

## Commands
- `npm install`: Install dependencies
- `npm run dev`: Run development server
- `npm run build`: Compile TypeScript
